# 致访客：
这仅仅是关于伪人（一种类型的小说）的资料库，不要信以为真，也不要随意破坏该仓库。后果自负。
# 介绍
该仓库的基本资料均为json格式，比如：
```json
{
    "saying":"hello!"
}
```
部分资料则采用txt格式、markdown格式或c++格式，如：
```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    cout<<"hello!";
    return 0;
}
```
txt:
```txt
hello!
```
markdown:
```markdown
# hello!
```
描述后续会更新更多内容。
